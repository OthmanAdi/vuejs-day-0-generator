# Der Hintergrund-Generator — Starter

**Alles zum Auftrag (Aufgabe · Beispiel · Wochenvorschau): https://sleek-vision-8ezb.here.now/**

Der Auftakt-Auftrag zum Vue.js-Modul (Dienstag 07.07.2026, Selbststudium): ein Generator, der endlose Marken-Hintergründe für deinen Klienten baut — Three.js r184, ein Fragment-Shader, eine HTML-Datei, kein Build.

## Losbauen in 60 Sekunden

```bash
git clone https://github.com/OthmanAdi/vuejs-day-0-generator.git
cd vuejs-day-0-generator
```

Dann `STARTER_generator.html` **per Doppelklick** im Browser öffnen. Du siehst einen Verlauf von Rot nach Grün — das ist Etappe 0, sie läuft schon. Kein Server, kein npm; nur die drei CDN-Zeilen der Importmap brauchen einmal Internet.

## Was hier liegt

| Datei | Was sie ist |

## Der Weg

1. `AUFGABE.md` lesen — zuerst die zwei Bausteine (~20 Min), die Artikel-Tabs offen lassen.
2. Etappe 1: Palette in den Shader, erster Zufall-Knopf.
3. Etappe 2: dein fBM aus dem Juni kehrt zurück.
4. Etappe 3: die Warp-Schleife — das Bild beginnt zu fließen.
5. Etappe 4: Slider, Zufall für alles, Fundstücke speichern.

Das fertige Referenz-Beispiel (alle Etappen + Druck-Ästhetik als Schalter) läuft auf https://sleek-vision-8ezb.here.now/beispiel.html — Code lesen erlaubt und erwünscht, der Marken-Pol muss deiner sein.
